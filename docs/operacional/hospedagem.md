# Hospedagem de Mídia (Media Hosting)

O modulo de **Hospedagem** é onde você centraliza todos os ativos visuais (imagens e vídeos) que serão utilizados nos disparos de WhatsApp.

## 📌 Biblioteca de Ativos
A página exibe uma grade de todos os arquivos hospedados no servidor do Plug & Sales, facilitando a reutilização de materiais entre diferentes campanhas.

## 🚀 Funcionalidades Principais

### 1. Upload Simples e em Massa
Você pode subir arquivos clicando no botão de upload ou simplesmente arrastando-os para a zona de drop. O sistema suporta múltiplos arquivos simultaneamente e exibe uma barra de progresso em tempo real.

### 2. Preview e Lightbox
Antes de usar uma imagem ou vídeo em um template, você pode clicar no ícone de **Maximizar** para abrir o arquivo em tela cheia (Lightbox). Isso permite conferir a qualidade e os detalhes do vídeo sem precisar baixar o arquivo.

### 3. Cópia de URL Otimizada
Cada arquivo possui um botão rápido para **Copiar URL**. 
- As URLs são otimizadas para carregamento rápido (CDN).
- O link copiado pode ser colado diretamente no campo de **Media URL** do Template Creator.

## ⚙️ Tecnologia Flow
O sistema utiliza tecnologia de ponta para garantir que suas mídias cheguem rápido ao celular do lead:
- **CDN Otimizada**: Carregamento ultra-rápido global, evitando que o vídeo "trave" ao ser aberto no WhatsApp.
- **Links Curtos**: URLs amigáveis e curtas para não comprometer o limite de caracteres dos templates.

## 💡 Dicas de Especialista
- **Organização por Busca**: Use a barra de busca para encontrar arquivos pelo nome. Uma boa prática é subir arquivos com nomes padronizados (ex: `campanha_natal_imagem01.png`).
- **Limpeza de Biblioteca**: Exclua arquivos antigos ou de testes para manter sua biblioteca limpa e fácil de navegar.
- **Preview de Vídeo**: No card de vídeo, você pode ver um pequeno preview em loop para identificar rapidamente o conteúdo sem abrir o lightbox.
